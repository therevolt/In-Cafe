    Erwin Saputra Sulistio - Fullstack Web 8, 
    Team Project - Week 8.
    
## How To Install?
- Clone This Repo
```
git clone https://github.com/ErwinSaputraSulistio/In-Cafe
```
- Go To Folder Repo
```
cd In-Cafe
```
- Install Module
```
npm install
```
- Type ``` npm start ``` To Start The Project
